นโยบายความเป็นส่วนตัว — 8เอ็กซ์เบท

วันที่อัปเดต: 19 มีนาคม 2569 (2026)

1. ภาพรวม

ส่วนขยาย Chrome "8เอ็กซ์เบท" ("ส่วนขยาย", "เรา") เป็นทางลัดที่ช่วยให้ผู้ใช้เข้าถึงลิงก์ เดิมพันกีฬา คาสิโนสด สล็อต และโปรโมชั่นของ 8เอ็กซ์เบท ได้รวดเร็วจากแถบเครื่องมือ Chrome เราให้ความสำคัญกับความเป็นส่วนตัวของคุณและมุ่งมั่นที่จะไม่เก็บข้อมูลส่วนบุคคลโดยไม่จำเป็น

สรุปสั้น: ส่วนขยายนี้ไม่ส่งข้อมูลใดๆ ออกไปยังเซิร์ฟเวอร์ภายนอก ข้อมูลทั้งหมดถูกจัดเก็บไว้ใน Chrome บนอุปกรณ์ของคุณเท่านั้น

2. ข้อมูลที่เก็บในอุปกรณ์

ส่วนขยายเก็บข้อมูลต่อไปนี้ไว้ใน Chrome Storage API บนอุปกรณ์ของคุณเท่านั้น:

  • รายการโปรด: รายการ ID ของลิงก์ที่คุณบันทึกไว้เป็นโปรด เพื่อแสดงผลในแผงรายการโปรด
  • ธีมสี: การตั้งค่าธีมสว่าง/เข้ม เพื่อแสดงผลตามความต้องการของคุณ

3. ข้อมูลที่ไม่มีการเก็บหรือส่ง

  • ข้อมูลส่วนตัว (ชื่อ ที่อยู่ อีเมล ฯลฯ)
  • ประวัติการเข้าชมเว็บไซต์
  • พฤติกรรมการคลิกหรือการใช้งาน
  • ข้อมูลทางการเงิน
  • ข้อมูลตำแหน่งที่ตั้ง
  • ข้อมูลอุปกรณ์หรือข้อมูลระบุตัวตน

4. การใช้สิทธิ์ (Permissions)

ส่วนขยายขอสิทธิ์เฉพาะที่จำเป็นเท่านั้น:

  • storage: ใช้สำหรับบันทึกรายการโปรดและธีมของคุณไว้ในอุปกรณ์ ไม่มีการส่งข้อมูลออกไปภายนอก

5. บริการเว็บไซต์ภายนอก

เมื่อคุณคลิกลิงก์ใดๆ ในส่วนขยาย คุณจะถูกนำไปยังเว็บไซต์ของ 8เอ็กซ์เบท (ukwelding.com หรือ 8xthb14.com) ซึ่งมีนโยบายความเป็นส่วนตัวของตนเอง เราแนะนำให้อ่านนโยบายดังกล่าวก่อนใช้งาน

6. ความปลอดภัย

ส่วนขยายไม่มีการรันโค้ดภายนอก (remote code execution) ไม่ใช้ eval() หรือ new Function() และทำงานสอดคล้องกับนโยบาย Manifest V3 ที่ Google กำหนด

7. การเปลี่ยนแปลงนโยบาย

หากมีการเปลี่ยนแปลงนโยบายนี้ เราจะอัปเดต "วันที่อัปเดต" ที่ด้านบนของหน้านี้ การใช้ส่วนขยายต่อหลังจากนั้นถือว่าคุณยอมรับนโยบายที่อัปเดตแล้ว
